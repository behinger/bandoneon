Help

This website wants to help learning the 142 button Bandoneon.

To internalize the button location on the both manuals the button map can be displayed.

  • The tabs above the keyboard switch the direction of the bellows (open/pull or close/push) and the side (left or right).
  • Select a key and a mode to display a scale
  • Select a key and a quality to display a chord (left side only!)

The keyboard is based on the 142 button Bandoneon ("Rheinische Tonlage")
